# Z Theme for Flarum

[![License: WTFPL](https://img.shields.io/badge/License-WTFPL-brightgreen.svg)](http://www.wtfpl.net/)

A beautiful and responsive theme for [Flarum](https://flarum.org/) forum software.

## Features

- 🎨 Modern and clean design
- 📱 Fully responsive layout
- ⚡ Lightweight and fast
- 🛠️ Easy to customize

## Installation

Install via Composer:

```bash
composer require zephyrisle/flarum-z-theme
```

Then enable the theme in your Flarum admin panel.

## Requirements

- Flarum Core ^1.0
